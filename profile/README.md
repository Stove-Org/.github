## 프로젝트 소개

> **스토브리그** : 하계 스포츠인 프로야구 정규 시즌이 끝난 이후에, 겨울 시즌 동안 중단되는 것에 착안해 팬들이 난로(Stove) 주위에 모여 선수단, 구단의 동향 등을 이야기하는 데에서 유래했습니다.

**Stove** 프로젝트는 e-스포츠 팬들이 스토브리그 기간을 더 재밌게 보낼 수 있는 플랫폼입니다.
